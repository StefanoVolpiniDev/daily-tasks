# 📝 Daily Tasks (To-Do List)

Um aplicativo de gerenciamento de tarefas prático e responsivo, desenvolvido com foco em **Clean Code** e manipulação eficiente do DOM. 

Este projeto foi construído para demonstrar habilidades sólidas nos fundamentos do Front-end e regras de negócio no lado do cliente, sem a utilização de frameworks.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização responsiva (Mobile-First) e interface limpa.
* **JavaScript (Vanilla / ES6+):** Lógica de programação, manipulação do DOM e eventos.

## ⚙️ Funcionalidades e Diferenciais

* **CRUD Completo:** Criação, leitura, edição e exclusão de tarefas.
* **Persistência de Dados:** Integração com a API de **LocalStorage** do navegador. As tarefas não são perdidas ao recarregar a página.
* **Filtros Inteligentes:** Opção de filtrar tarefas por status (Todas, Feitas, A Fazer).
* **Busca em Tempo Real:** Barra de pesquisa que filtra as tarefas instantaneamente conforme a digitação.

## 💻 Como executar o projeto

Como o projeto não utiliza dependências externas ou servidores backend, a execução é extremamente simples:

1. Faça o clone deste repositório ou baixe o arquivo ZIP.
2. Extraia os arquivos para uma pasta.
3. Abra o arquivo `index.html` em qualquer navegador web de sua preferência.
